# Smart Home Documentación

## 🔍Objetivos🔎 

El objetivo es desarrollar una página web para el control de una casa inteligente. El proyecto debe incluir un "modo noche" y un modo día" gestionado únicamente con variables.

## 🎯Requisitos

- No se permite el uso de píxeles (px) para font-size, padding, margin o anchos de contenedor.
- Deben usar rem para tipografía y espaciados, y em para componentes contenidos. Los anchos deben ser en %, vw/vh o fracciones de grid (fr)
- No se puede definir cada celda manualmente.
- El dashboard debe usar grid-template-columns: repeat(auto-fill, minmax(300px, 1fr)) en al menos una sección.
-  Deben crear un sistema de capas de abstracción para las variables.
-  El proyecto debe construirse bajo la filosofía Mobile-First. El CSS base debe ser la versión móvil, y las Media Queries deben usar exclusivamente min-width.

## 💻Tecnologias

La tecnologia usada es la siguiente:
- Para el desarrollo inicial y la elaboracion del prototipo hemos usado la herramienta Figma.
- Para el desarrollo de la página web hemos usado tecnologias como HTML:5 y CSS

## 🖥️¿Como ejecutarlo?
Para ejecutarlo, debemos tener el id Visual Studio Code, con la extension Live Server. Esto hara que podamos editar el codigo y ver el estado de la página web.
